# Loft Price Index  (LPI)

Анализ динамики цен на **лофты** и **посвяты** с сопоставлением с макроэкономическими показателями РФ (инфляция, ключевая ставка).

---

## Описание проекта

Цель проекта — понять, как менялись цены на мероприятия и как они соотносятся с макроэкономикой:

- обгоняют ли цены инфляцию  
- есть ли влияние ключевой ставки  
- произошёл ли структурный сдвиг после 2023 года  

---

## Данные

### Пользовательские данные (VK)

Источник: https://vk.com/ikanam  

**Поля:**
- `date` — дата события  
- `year`, `month`  
- `type` — `loft` / `posvyat`  
- `min_price` — минимальная цена  
- `avg_price` — средняя цена  
- `link` — ссылка на пост  

---

### Макроэкономика

- ИПЦ (инфляция):  
  https://cbr.ru/statistics/macro_itm/inflation/  
  или https://rosstat.gov.ru/statistics/price  

- Ключевая ставка:  
  https://cbr.ru/hd_base/keyrate/

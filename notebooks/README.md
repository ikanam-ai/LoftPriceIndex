# Ноутбуки: анализ цен на лофты и посвяты

## Структура

| Ноутбук | Описание |
|---|---|
| `00_exploratory_data_analysis.ipynb` | Первичный разведочный анализ: загрузка данных, распределение цен, визуализация динамики по годам |
| `01_indexes.ipynb` | Индексы цен, сравнение с инфляцией (ИПЦ), реальные цены, связь с ключевой ставкой ЦБ РФ |
| `02_econometrics_loft.ipynb` | Эконометрика: тренды, стационарность, макро-корреляции, ARIMA-прогноз, точка пересечения реальных цен |

## Источники данных

Все данные хранятся локально в папке `../data/`:

| Файл | Содержимое | Источник |
|---|---|---|
| `Лофты посвяты.xlsx` | Исторические данные о ценах на мероприятия (лофты и посвяты) | Собственный сбор |
| `ИПЦ.xlsx` | Индекс потребительских цен РФ по годам | [rosstat.gov.ru](https://rosstat.gov.ru/statistics/price) |
| `ключевая ставка.xlsx` | Дневные значения ключевой ставки ЦБ РФ | [cbr.ru](https://www.cbr.ru/hd_base/keyrate/) |
| `macro.csv` | ИПЦ по категориям (алкоголь, культура, аренда) и средняя ставка ЦБ по годам | Сводная таблица на основе данных Росстат и ЦБ РФ |

## Ключевые выводы (ноутбук 02)

### Тренды цен
- **Лофты** растут примерно на **+157 ₽ в год** (R² = 0.894, тренд значим)
- **Посвяты** растут примерно на **+106 ₽ в год** (R² = 0.593, тренд значим)

### До и после 2023 года
- Лофты: темп роста ускорился с **+70 ₽/год** → **+281 ₽/год** (Δ = +211 ₽/год)
- Посвяты: темп роста изменился с **+196 ₽/год** → **−108 ₽/год** (разворот тренда)

### Стационарность
Оба ряда дают смешанные результаты ADF/KPSS — нестационарность нельзя исключить, интерпретация ARIMA выполняется с осторожностью.

### Макро-связи (на темпах роста)
- Лофты: значимая положительная корреляция с **ключевой ставкой** (r = +0.839, p = 0.037)
- Посвяты: значимых корреляций с макро-показателями не обнаружено

### ARIMA-прогноз (лучшая модель — (1,1,0))
| Тип | MAPE | Прогноз +1 год | Прогноз +3 года |
|---|---|---|---|
| Лофты | 6.0% | ≈ 2 192 ₽ | ≈ 2 513 ₽ |
| Посвяты | 6.8% | ≈ 3 699 ₽ | ≈ 3 695 ₽ |

### Реальные цены (база 2019 г.)
С 2022 года реальные цены лофтов растут (+97.8 ₽/год), посвятов — падают (−202.4 ₽/год).  
Оценочная точка пересечения трендов — около **2030 года** при цене ≈ 1 527 ₽ (в ценах 2019 г.).
